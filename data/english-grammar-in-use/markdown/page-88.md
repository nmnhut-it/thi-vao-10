# Unit 38 - if I do ... and if I did ...

## Section A - First Conditional vs Second Conditional

### First Conditional: if + present, ... will ...
Real/possible situation:
- **If** we **take** the bus, it **will** be cheaper. (it is possible that they will take the bus)

### Second Conditional: if + past, ... would ...
Imagined/unlikely situation:
- **If** we **took** the bus, it **would** be cheaper. (they are NOT going to take the bus)

## Section B - Using if + past for imagined situations

When we talk about something that will not happen or we don't expect to happen, we use **if + past**:
- What would you do **if** you **won** a lot of money? (I don't expect this to happen)
- **If** there **was** an election tomorrow, who would you vote for? (there will not be an election)
- I'd be surprised **if** they **didn't come** to the party. (I expect them to come)

### Compare if I find and if I found:
- I think I left my watch at your house. **If** you **find** it, can you call me? (real possibility)
- **If** you **found** a wallet in the street, what would you do with it? (imagined)

## Section C - would in the if-clause

We do NOT normally use **would** in the if-part:
- I'd be very scared **if** somebody **pointed** a gun at me. (*not* if somebody would point)
- **If** we **went** by bus, it would be cheaper. (*not* if we would go)

**would** goes in the other part of the sentence:
- I'**d be** scared if somebody pointed a gun at me.
- What **would** you **do** if you were bitten by a snake?

### could and might are also possible in the result clause:
- If I won a lot of money, I **might** buy a house.
- If it stopped raining, we **could** go out.
