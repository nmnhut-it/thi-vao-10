# Unit 49 - Exercises: Questions 1

## 49.1 Ask Joe questions.
1. (where / live) **Where do you live?** → In Manchester.
2. (born there?) ___ → No, I was born in London.
3. (married?) ___ → Yes.
4. (how long?) ___ → 17 years.
5. (what / do?) ___ → I'm a journalist.
6. (what wife / do?) ___ → She's a doctor.
7. (children?) ___ → Yes, two boys.
8. (how old?) ___ → 12 and 15.
