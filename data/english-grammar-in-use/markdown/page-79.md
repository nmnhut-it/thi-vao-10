# Unit 33 - Exercises: should 1

## Exercise 33.1 - Write should or shouldn't + phrase from box
Phrases: go away for a few days, put some pictures on the walls, stay up so late, look for another job, take a picture, worry so much

1. Anna needs a change. **She should go away for a few days.**
2-6. Write advice sentences using should/shouldn't.

## Exercise 33.2 - Complete with should from box
Options: should solve, should be working OK, should pass the exam, shouldn't cost more, shouldn't take long, should be here soon, should be much warmer, shouldn't be more than ten pounds

1. Helen has been studying hard, so she **should pass the exam**.
2-8. Complete sentences expressing expectations.

## Exercise 33.3 - Complete with should or should have + verb
1. You look tired. You **should go** to bed. (go)
2. You missed a great party. **You should have come.** (come)
3-8. Distinguish between present advice (should do) and past regret (should have done).

## Exercise 33.4 - Read situations and write sentences
Use should / should have / shouldn't / shouldn't have.

1. I'm feeling sick. I ate too much. - **I shouldn't have eaten so much.**
2. When we got to the restaurant, there were no free tables. We hadn't reserved one. - We ___
3. Laura told me her address, but I didn't write it down. - I ___
4-9. Continue writing should/shouldn't sentences based on situations.
