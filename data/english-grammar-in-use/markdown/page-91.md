# Unit 39 - Exercises: if I knew ... I wish I knew ...

## Exercise 39.1 - Put the verb into the correct form
1. If **I knew** (I / know) his number, I would phone him.
2. **I wouldn't buy** (I / not / buy) that coat if I were you.
3. ___ (I / help) you if I could, but I'm afraid I can't.
4. This soup isn't very nice. ___ (it / taste) better if it wasn't so salty.
5-12. Complete second conditional and wish sentences with correct verb forms.

## Exercise 39.2 - Write a sentence with if for each situation
1. We don't see you very often because you live so far away. - **If you didn't live so far away, we'd see you more often.**
2. I like these shoes but they're too expensive, so I'm not going to buy them.
3. We'd like to go on holiday, but we can't afford it.
4-5. Continue writing if-sentences from situations.

## Exercise 39.3 - Write sentences beginning I wish
Express regret about current situations.

1. I don't know many people (and I'm lonely). **I wish I knew more people.**
2. I don't have much free time (and I need more). I wish ___
3. Helen isn't here (and I need to see her). ___
4-9. Continue writing I wish sentences.

## Exercise 39.4 - Write your own sentences beginning I wish
Personal sentences about:
1. Somewhere you'd like to be now
2. Something you'd like to have
3. Something you'd like to be able to do
4. Something you'd like to be
